# Copa do Mundo 2026 - Tabela

Este projeto é uma página web estática desenvolvida em React para exibir a tabela da Copa do Mundo de 2026.

## Funcionalidades

- **Barra Lateral**: Lista de países classificados (simulados e confirmados).
- **Tabela de Grupos**: Exibição dos grupos com pontuação e estatísticas.
- **Design**: Layout responsivo com fundo temático da Copa de 2026.

## Tecnologias

- React
- Vite
- CSS

## Como rodar o projeto

1. Instale as dependências:

    ```bash
    npm install
    ```

2. Inicie o servidor de desenvolvimento:

    ```bash
    npm run dev
    ```

3. Acesse `http://localhost:5173` no seu navegador.

## Deploy no GitHub Pages

Para fazer o deploy no GitHub Pages, configure o `base` no `vite.config.js` e use uma action de deploy ou o pacote `gh-pages`.

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
